\# NOOB ZOMBIE SHOOTER



\## Game Overview



NOOB ZOMBIE SHOOTER is a first-person zombie survival shooter where the player navigates through a maze while fighting off zombies. The objective is to survive and eliminate zombies while exploring the environment.



The game features multiple weapons and immersive first-person controls designed for fast-paced gameplay.



\---



\## Controls



| Key          | Action                |

| ------------ | --------------------- |

| W        | Move Forward          |

| S        | Move Backward         |

| A        | Move Left             |

| D        | Move Right            |

| Spacebar | Jump                  |

| R        | Reload Weapon         |

| 1        | Equip Desert Eagle    |

| 2        | Equip MP5             |

| 3        | Switch to Normal Hand |



\---



\## Weapons



\### Desert Eagle



A powerful handgun with high damage.



\### MP5



A rapid-fire submachine gun useful for taking down multiple zombies quickly.



\### Normal Hand



Allows the player to move without holding a weapon.



\---



\## Enemies



The zombies in the game were created using Tripo AI, providing detailed 3D models for enemies and weapons.



\---



\## Objective



Survive in the environment and eliminate zombies using the available weapons while navigating the environment.



\---



\## Requirements



\* Windows PC

\* Keyboard and Mouse

\* Unity-supported graphics hardware



\---



\## How to Run the Game
1. Extract the zip files.

2\. Open the game folder.

3\. Ensure the following files remain in the same directory:



&#x20;  \* NOOBZOMBIESHOOTER.exe

&#x20;  \* NOOBZOMBIESHOOTER\_Data

&#x20;  \* UnityPlayer.dll

4\. Double-click \*\*NOOBZOMBIESHOOTER.exe\*\* to start the game.
